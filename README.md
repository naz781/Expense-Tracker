# Expense Tracker

A simple **Expense Tracker** built with **React** and **Vite** to help you manage your personal expenses. Track your income, expenses, and get a quick overview of your spending habits.

## Features

- Add, edit, and delete expenses  
- View a list of all transactions  
- Categorize your expenses  
- Track income vs. expenses  
- Responsive UI for both desktop and mobile

> Note: Currently, the app stores data in **local storage**. No backend integration yet.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/naz781/Expense-Tracker.git
cd Expense-Tracker
```
##Install dependencies:

```bash
npm install
```

## Start the development server:

```bash
npm run dev

```
Open http://localhost:5173
 in your browser.

## Usage

1.Open the app in your browser.

2.Add a new transaction by entering a description, amount, and type (income or expense).

3.Edit or delete transactions as needed.

4.View the updated balance and transaction list in real-time.

## Technologies Used

* React – Frontend library

* Vite – Build tool & development server

* JavaScript (ES6+) – Core language

* HTML5 & CSS3 – Markup and styling

* LocalStorage – Temporary data persistence

* FireBase - for database and Authentication 

(Future versions may include backend integration for persistent storage)
